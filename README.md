# Global Forest Data Analysis (1990–2020)

## Project Objective

This project analyzes global forest data from 1990 to 2020 in order to better understand how forest area is distributed across continents, how it has changed over time, and what these patterns suggest about global environmental trends.

Rather than simply presenting the dataset, the project applies a question-driven analytical approach to explore forest distribution, regional differences, and long-term change through data cleaning, visualization, and interpretation.

---

## Dataset

- **Source:** Kaggle

- **Dataset:** 1990–2020 FAO Global Forest Dataset

- **Format:** CSV

This dataset includes country-level forest indicators such as:

- Forest area across multiple years

- Total land area

- Forest-to-land ratio

- Deforestation-related indicators

- Fire-affected forest area

---

## Project Structure

### 1. Data Acquisition

- Imported the dataset into Python using Jupyter Notebook

- Loaded and inspected the raw CSV structure

### 2. Data Cleaning and Preparation

- Renamed columns into a clearer and more consistent format

- Identified missing values and invalid rows

- Converted inconsistently formatted numeric columns into usable numerical data types

- Prepared the dataset for reliable analysis

### 3. Exploratory Data Analysis (EDA)

- Examined the structure and quality of the dataset

- Reviewed distributions, missing values, and data types

- Identified early patterns and potential analytical directions

### 4. Regional Forest Analysis

- Grouped countries by continent

- Compared average forest area and forest-to-land ratio across continents

- Explored how regional scale affects the interpretation of forest statistics

### 5. Temporal Analysis

- Analyzed changes in forest area across continents from 1990 to 2020

- Calculated net percentage change over time

- Examined long-term continental trends through line charts and comparative summaries

### 6. Visualization and Interpretation

The project includes several visualizations to support interpretation:

- **Bar Chart:** Comparison of forest coverage ratios across continents

- **Line Chart:** Change in forest area by continent over time

- **Diverging Bar Chart:** Net forest change by continent (1990–2020)

- **Donut Charts:** Global forest distribution by continent in 1990 and 2020

These visualizations were used not only to display the data, but also to support analytical interpretation and highlight broader environmental patterns.

### 7. Conclusion

The project shows that global forest distribution remained broadly stable in structure, but regional changes over time reveal important differences in environmental dynamics. It also highlights the importance of using both absolute and relative measures when interpreting forest data.

---

## Analytical Approach

This project was developed using a layered analytical structure:

- **Geographical Analysis**  

  Comparing forest area and forest coverage across continents

- **Temporal Analysis**  

  Examining how forest area changed between 1990 and 2020

- **Comparative Interpretation**  

  Combining regional and temporal perspectives to produce more meaningful conclusions

This approach helped turn raw environmental data into a more structured and interpretable analysis.

## Key Takeaways

- Forest area is unevenly distributed across continents

- Large forest area does not always mean high forest density

- Regional forest trends differ significantly over time

- Long-term environmental change is better understood when geographic and temporal analysis are combined
